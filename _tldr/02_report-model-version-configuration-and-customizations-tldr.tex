Researchers \must report the exact LLM model or tool version, configuration, and experiment date in the \paper. When using quantized models, researchers \should report the quantization level and method. For fine-tuned models, they \must describe the fine-tuning goal, dataset, and procedure in the \paper. Researchers \should include default parameters, explain model choices, compare base- and fine-tuned model using suitable metrics and benchmarks, and share fine-tuning data and weights as \supplementarymaterial (or alternatively justify in the \paper why they cannot share them).
